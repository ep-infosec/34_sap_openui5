Integration Card Types

Declarative Card Types

This section describes the different content types that can be used in the card content area ("sap.card".content in the manifest).

Declarative cards are preferable for consistency, performance and maintenance aspects.
Usage of those cards should be also considered for scenarios that require a native mobile (iOS/Android) experience in the future.

OpenUI5 vs SAPUI5

Depending on the UI5 delivery you are using (SAPUI5 or OpenUI5) there are different declarative card types:

OpenUI5

  • List
  • Object
  • Table
  • Calendar

SAPUI5 - adds Timeline and Analytical

  • Analytical
  • Timeline
